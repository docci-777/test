# 04 - 战斗与撤离系统

> 本文件定义战斗触发、结算公式、撤离路径劫掠与掉落机制。
> 战斗是连接烽火搜刮与边防攻防的枢纽。

---

## 1. 战斗类型

| 类型 | 触发场景 | 说明 |
|---|---|---|
| 边防攻城战 | 攻方移动进入敌方关隘边 | 强攻防线 |
| 劫掠战 | 撤离队经过敌方控制边 | 烽火核心冲突 |
| 禁区遭遇战 | 禁区内两支玩家搜刮队相邻 | 舔包机会 |
| PvE 战 | 与 AI 蚀体战斗 | 搜刮阻碍 |
| 据点围困战 | 攻方围困敌方据点 | 终局手段 |

## 2. 单位属性

### 2.1 搜刮队属性
| 属性 | 说明 |
|---|---|
| 生命值 | 归零则全灭，装备物资全丢 |
| 火力 | 由武器决定，影响伤害 |
| 护甲 | 由护甲决定，影响减伤 |
| 弹药 | 战斗消耗，耗尽则火力降级 |
| 负重 | 影响禁区行动点与撤退速度 |
| 士气 | 受连续战斗/损血影响，低士气减属性 |

### 2.2 边防军属性
| 属性 | 说明 |
|---|---|
| 兵力 | 多单位叠加，归零则溃散 |
| 攻防 | 基础值 + 建筑加成 |
| 驻防加成 | 在己方边建筑上战斗时获得 |

## 3. 战斗结算公式

### 3.1 基础伤害
```
伤害 = 火力 × 武器系数 × 骰子(0.8–1.2) × 地形修正
实际伤害 = 伤害 × (1 - 护甲减伤率)
```

### 3.2 骰子机制
- 每次战斗掷 1 颗骰子（1–6），映射为 0.8 / 0.9 / 1.0 / 1.0 / 1.1 / 1.2 倍率。
- 数字版可用随机数替代，桌游版用实体骰。

### 3.3 地形与建筑修正

| 情形 | 修正 |
|---|---|
| 守方在关隘上 | 伤害 -30%（受庇护），攻方伤害 -20% |
| 守方在哨塔范围 | 守方先手攻击 |
| 伏击点触发 | 伏击方先手 + 首回合伤害 +50% |
| 河流/山地交战 | 双方伤害 -20% |
| 撤离队负重满 | 撤离队伤害 -15%，移动力 -1 |

### 3.4 弹药消耗
- 每次战斗消耗 1 弹药。
- 弹药耗尽 → 火力降级为副武器/近战水平。
- 弹药箱可补充。

## 4. 撤离路径劫掠（核心机制）

### 4.1 路径规则
- 搜刮队撤离出禁区镇后，玩家指定**返回据点的六边形边路径**。
- 路径长度受搜刮队移动力限制（负重影响）。
- 路径上每条**敌方控制边**依次结算。

### 4.2 劫掠战流程
1. 撤离队进入敌方控制边 → 触发劫掠战。
2. 守方（劫掠者）有建筑加成 + 以逸待劳。
3. 战斗结算（见第 3 节）。
4. **撤离队胜** → 通过该边，继续下一条边。
5. **撤离队败** → 装备物资全被劫（保险箱内除外），搜刮队溃散回据点（不死亡，但本局搜刮成果归零）。
6. **撤离队全灭** → 装备物资全丢，搜刮队需下回合重新组建。

### 4.3 多重劫掠
- 路径上可能连续遭遇多个敌方边防。
- 撤离队每场战斗后状态不回满，形成"层层剥皮"。
- 因此**预先清剿回家路**是高阶策略。

### 4.4 反劫掠
- 撤离队可携带战斗向装备（步枪/手雷）应对劫掠。
- 也可派**护镖边防军**提前到路径关键边接应（需行动阶段调度）。

## 5. 掉落与舔包

### 5.1 搜刮队全灭掉落
- 装备 + 背包物资全部掉落于该边/节点。
- 保险箱内物品保留退回军械库。
- 掉落物可被**胜方当场拾取**，或留作**公共拾取点**（其他玩家后续可捡）。

### 5.2 边防军溃散
- 边防军溃散不掉落装备（边防军无装备系统）。
- 但其驻守的**边建筑可能被占领或摧毁**。
  - 占领：攻方获得该边控制权。
  - 摧毁：建筑消失，边变中立（攻方选择）。

## 6. 战斗先手与回合内顺序

- 同一回合内多场战斗按**触发顺序**结算。
- 伏击点/哨塔方**始终先手**。
- 其他情况**守方先手**（防御优势）。
- 禁区遭遇战双方同时结算（互伤）。

## 7. 士气系统（可选，建议 MVP 简化）

- 连续战斗 / 损血过半 → 士气下降。
- 低士气 → 火力 -10% / 溃散概率提升。
- 医疗包/补给站可恢复士气。
- MVP 阶段可先不实装，后续扩展。

## 8. 围困与破围

### 8.1 围困
- 据点所有相邻边被敌方关隘封锁 → 该玩家**无法派出搜刮队**。
- 围困期间仍可收取据点基础产出，但无法入禁区。

### 8.2 破围
- 被围困玩家需用边防军强攻任一相邻关隘。
- 破围成功 → 恢复派出能力。
- 长期围困是终局淘汰手段之一。

## 9. 战斗结果对局外的影响

| 结果 | 局外影响 |
|---|---|
| 搜刮队成功撤离 | 物资入库，军械库装备保留 |
| 搜刮队被劫 | 物资归零，装备丢失（保险箱除外） |
| 搜刮队全灭 | 物资装备全丢（保险箱除外） |
| 边防军溃散 | 边建筑被占/毁，需重建 |
| 劫掠成功 | 获得对方掉落物资装备 |

> 装备丢失是烽火式惩罚的核心，制造"高风险高回报"张力。
