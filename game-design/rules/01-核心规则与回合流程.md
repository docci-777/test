# 01 - 核心规则与回合流程

> 本文件定义游戏的基础框架、玩家身份、资源体系与单回合六阶段流程。
> 其他规则文件在此基础上展开。

---

## 1. 游戏概述

- **名称**：《灰烬边境：烽火六镇》
- **类型**：2D 平面、六边形网格、回合制策略 + 烽火撤离搜刮
- **人数**：2–6 人（推荐 4–6 人）
- **单局时长目标**：60–120 分钟（数字版可加速结算）
- **核心循环**：局外养成 → 局前构筑 → 边防建设 → 禁区搜刮 → 撤离劫镖 → 回局外养成

## 2. 玩家身份

每位玩家扮演一支**拓荒团**，拥有：
- 1 个**据点**（出生地，不可被摧毁，可被围困）
- 1 个**军械库**（局外养成，决定可携带装备池）
- 1 个**势力天赋**（见 `05-势力与局外养成.md`）
- 若干**单位**：搜刮队 + 边防军

### 2.1 单位分类

| 单位 | 用途 | 备注 |
|---|---|---|
| 搜刮队 | 进入禁区搜刮、撤离搬运 | 局前需配置装备 |
| 边防军 | 驻守边防、劫掠撤离队 | 不可进入禁区 |
| 工兵 | 建造/升级边防建筑 | 无战斗力 |

> MVP 阶段每类单位各 1 队起步，随养成扩充。

## 3. 资源体系

三种主资源 + 一种稀有资源：

| 资源 | 产出 | 用途 |
|---|---|---|
| 资材 | 据点 + 控制地块 | 建造边防、招募单位 |
| 弹药 | 据点 + 补给站 | 禁区内战斗消耗 |
| 灰核 | 禁区搜刮 + 祭坛 | 升级军械库、科技树、终局胜利 |
| 口粮（可选） | 据点 | 维持单位上限，避免无限爆兵 |

- 每回合筹措阶段按**控制地块数 + 据点等级**结算产出。
- 灰核**只能从禁区带出**，是稀缺战略资源。

## 4. 回合制结构

每**轮**所有玩家依次完成自己的**回合**。单回合分六个阶段，按顺序执行：

### 阶段 1：筹措阶段
- 收取本回合资源产出（资材/弹药/灰核）。
- 结算持续效果（建筑产出、debuff 等）。

### 阶段 2：整备阶段
- 为即将派出的搜刮队配置装备（局前构筑，见 `03-烽火摸金玩法.md`）。
- 配置一旦确认，本回合不可更改。
- 未派出搜刮队可跳过。

### 阶段 3：行动阶段
- 移动单位（每单位有移动力，受地形影响）。
- 工兵建造/升级边防建筑（见 `02-地图与边建造系统.md`）。
- 调整边防军驻防位置。
- **不可**在此阶段进入禁区（禁区进入专属烽火阶段）。

### 阶段 4：烽火阶段
- 已整备的搜刮队进入禁区镇，执行搜刮与战斗（见 `03-烽火摸金玩法.md`）。
- 一支搜刮队每回合最多进入一次禁区。
- 多支搜刮队可分别进入不同禁区镇。

### 阶段 5：撤离阶段
- 禁区内搜刮队选择撤离点呼叫撤离，或继续停留（下回合再撤）。
- 已撤离出禁区的搜刮队沿六边形边路径返回据点。
- 路径若被敌方边防封锁 → 触发劫掠战（见 `04-战斗与撤离系统.md`）。

### 阶段 6：结算阶段
- 战斗最终结算。
- 触发全局/剧情事件（见 `06-主线剧情与胜利条件.md`）。
- 检查胜利条件。
- 轮到下一位玩家。

## 5. 关键约束

- **行动点限制**：每阶段有独立行动点预算，不可跨阶段挪用。
- **建造与移动互斥**：工兵一回合内建造与移动二选一（升级除外）。
- **禁区停留上限**：搜刮队连续停留禁区超过 N 回合触发"灰蚀侵蚀"，每回合损血。
- **围困机制**：据点所有相邻边被敌方关隘封锁时，该玩家无法派出搜刮队，需先破围。

## 6. 玩家顺序与先手

- 第一轮随机决定顺序。
- 之后每轮按**当前灰核持有量升序**行动（落后者先手，抑制滚雪球）。
- 灰核相同则保持上轮顺序。

## 7. 待定参数（需测试调优）

| 参数 | 初始建议 | 说明 |
|---|---|---|
| 每回合行动点 | 3 | 行动阶段用 |

| 搜刮队禁区行动点 | 4 | 受负重影响 ±2 |
| 禁区停留上限 | 3 回合 | 超过触发侵蚀 |
