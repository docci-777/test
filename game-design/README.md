## 阅读建议

- **想了解怎么玩**：按 rules 01 → 06 顺序阅读。
- **想做开发决策**：先读 dev-boundaries 01–02，再读 04 MVP 定义。
- **想做技术选型**：读 dev-boundaries 03。
- **想快速验证可行性**：读 dev-boundaries 04 的 MVP 验收标准。

## 核心设计一句话

> 搜刮产出驱动边防建设，边防建设反过来劫掠搜刮——烽火撤离与六边形攻防在"撤离路径"上闭环。
