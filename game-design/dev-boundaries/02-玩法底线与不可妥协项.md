# 02 - 玩法底线与不可妥协项

> 本文件定义游戏必须保留的核心体验、不可妥协的机制与平衡底线。
> 任何版本迭代、功能裁剪或简化都不得违反本文件。

---

## 1. 不可妥协的核心体验

以下五项是游戏灵魂，缺失任一即不再是本游戏：

### 1.1 "带得出来才算赢"
- 搜刮成功 ≠ 获胜，必须撤离成功才算归属。
- 撤离路径存在被劫风险。
- **底线**：任何版本都必须保留"搜刮 → 撤离 → 劫掠"完整闭环。

### 1.2 "装备即资产"
- 装备是局外积累的资产，带入禁区有丢失风险。
- 全灭/被劫 = 装备丢失（保险箱除外）。
- **底线**：装备丢失惩罚不可移除，否则烽火张力消失。

### 1.3 "边建造卡道劫镖"
- 边建造的核心用途之一是劫掠撤离队。
- 玩家可在对手回家路上建关隘蹲守。
- **底线**：边建造与撤离路径必须物理耦合，不可解耦。

### 1.4 "构筑流派博弈"
- 局前战备挑选必须支持多种可行流派。
- 全装/轻装/平衡/保命至少 4 流派可用。
- **底线**：战备点 + 负重双重限制不可移除。

### 1.5 "多路径胜利"
- 至少 4 条胜利路径同时存在。
- 不可削减为单一胜利条件。
- **底线**：灰核归一为主线，但其他路径必须真实可行。

## 2. 不可妥协的机制

| 机制 | 不可移除原因 |
|---|---|
| 六边形边建造 | 攻防骨架，用户明确要求 |
| 局前战备挑选 | 烽火核心，用户明确要求 |
| 局内带物资撤离 | 烽火核心，用户明确要求 |
| 回合制 | 用户明确要求 |
| 6 人支持 | 用户明确要求上限 |
| 主线剧情 | 用户明确要求 |
| 保险箱 | 平衡"保命保货"与"贪心多带" |
| 暂存点 | 平衡背包容量与多轮搜刮 |
| 落后者先手 | 滚雪球抑制核心 |

## 3. 平衡底线

### 3.1 资源底线
- 灰核**只能从禁区带出**，不可通过边防/据点稳定产出（祭坛除外，且祭坛造价含灰核）。
- 资材产出有上限，防止无限爆兵。
- 弹药消耗存在，防止无限战斗。

### 3.2 战斗底线
- 守方有防御优势（先手/减伤），但攻方可通过数量/装备/伏击反制。
- 不存在"无敌组合"，任何配置都有克制手段。
- 撤离队负重满时必须有明确劣势（减伤/减速）。

### 3.3 养成底线
- 军械库最高等级不可一步到位，必须多局积累。
- 装备丢失后可重新生产，但消耗资源（不可免费复活）。
- 科技树三大分支不可同时满级（灰核稀缺约束）。

### 3.4 节奏底线
- 单局总轮数 ≥ 12（保证四幕剧情完整）。
- 前期（1–3 轮）不可出现终局级冲突（教学期保护）。
- 终局（12 轮+）必须有明确收束机制（灰核归一决战）。

## 4. 反滚雪球底线（强制机制）

以下机制是滚雪球抑制的硬性保障，不可移除：

| 机制 | 作用 |
|---|---|
| 落后者先手（按灰核升序） | 落后者优先行动 |
| 装备丢失惩罚 | 强队全灭大伤元气 |
| 劫镖翻盘 | 一夜暴富可能 |
| PvE 蚀体压力 | 分散强者注意力 |
| 碎片全图可见 | 强者成众矢之的 |
| 围困可破 | 不可一劳永逸困死对手 |

## 5. 公平性底线

- 所有玩家初始资源相同。
- 势力天赋差异不导致碾压（需测试调优）。
- 地图出生点对称（六角分布，资源均衡）。
- 随机性（骰子/物资分布）不可决定胜负，只影响单次战斗/搜刮。

## 6. 不可引入的设计

| 禁止项 | 原因 |
|---|---|
| 付费变强 | 破坏公平 |
| 永久死亡惩罚（局外） | 过度惩罚，劝退 |
| 无限资源机制 | 破坏稀缺性 |
| 不可逆劣势 | 违反滚雪球抑制 |
| 单一最优策略 | 破坏多路径胜利 |
| 实时操作元素 | 违反回合制定位 |

## 7. 简化时的保留优先级

若需简化（如 MVP 或新手模式），按以下优先级保留：
1. 撤离劫掠闭环（最高优先）
2. 局前战备构筑
3. 边建造系统
4. 多胜利条件
5. 局外养成
6. 六势力差异化
7. 剧情事件
8. 士气/暂存等辅助机制（可暂缓）
