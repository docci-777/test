# 04 - 内容范围与 MVP 定义

> 本文件定义 MVP（最小可行版本）内容范围、后续扩展路线与优先级。
> 用于控制开发范围，确保核心玩法可验证。

---

## 1. MVP 目标

**验证核心假设**：烽火撤离搜刮 × 六边形边建造攻防的耦合是否好玩。

**MVP 不追求**：完整剧情、全部势力、全部装备、平衡性完美。

**MVP 必须达成**：双玩法闭环可玩、撤离劫掠有张力、构筑流派有取舍。

## 2. MVP 内容范围

### 2.1 玩家与势力
- 人数：支持 **2–4 人**（6 人留扩展）。
- 势力：**3 个**（摸金校尉/卸岭力士/观山太保，差异化明显）。
- 势力技能：被动 + 主动各 1，简化实现。

### 2.2 地图
- 尺寸：半径 4 的六边形地图（约 61 格）。
- 禁区镇：**3 座**（中央）。
- 地形：平原、山地、密林、河流、聚落、禁区镇（6 种）。
- 出生点：3–4 个对称分布。

### 2.3 边建造
- 建筑种类：**6 种**（关隘、哨塔、陷阱带、补给站、伏击点、采掘场）。
- 升级：关隘支持 3 级，其余暂不升级。
- 暗道/祭坛/军械所等留扩展。

### 2.4 装备库
- 装备种类：**12 种**（覆盖 4 流派）。
  - 武器：手枪、冲锋枪、步枪、霰弹枪
  - 护甲：轻甲、重甲
  - 背包：战术包、扩容包
  - 消耗品：医疗包、弹药箱、手雷
  - 特殊：保险箱
- 军械库等级：Lv1–Lv3。
- 战备点：20，负重系统启用。

### 2.5 禁区镇
- 节点数：每镇 8–10 个。
- 节点类型：物资箱、武器架、灰核矿、陷阱、AI 蚀体巢、撤离点。
- 撤离点：每镇 2 个。
- 暂存点：每镇 1 个。
- 保险箱：启用。

### 2.6 战斗
- 基础伤害公式 + 骰子。
- 弹药消耗。
- 地形/建筑修正（关隘减伤、伏击先手）。
- 士气系统：**MVP 不实装**。

### 2.7 撤离劫掠
- 撤离路径规划。
- 路径上敌方边触发劫掠战。
- 全灭掉落 + 保险箱保留。
- 舔包机制。

### 2.8 养成
- 军械库 Lv1–3。
- 据点 Lv1–3。
- 科技树：**MVP 不实装**（灰核仅用于升级军械库/据点）。

### 2.9 剧情
- **MVP 简化**：不实装四幕剧情与全局事件。
- 胜利条件：仅保留 2 条（灰核归一简化版 + 歼灭）。
- 灰核碎片：3 块（对应 3 禁区镇），集齐护送回据点即胜。

### 2.10 AI
- 蚀体：游荡者 + 守巢者 2 种，规则驱动。
- 人机对手：**MVP 可选**，简单规则 AI 即可。

## 3. MVP 排除项（明确不做）

| 排除项 | 原因 |
|---|---|
| 6 人支持 | MVP 验证 2–4 人即可 |
| 全部 6 势力 | 3 势力足够验证差异化 |
| 全部装备/建筑 | 精简到验证流派即可 |
| 四幕剧情 | MVP 聚焦玩法，剧情后补 |
| 全局事件 | 增加复杂度，后补 |
| 科技树 | 灰核用途简化 |
| 士气系统 | 辅助机制，后补 |
| 暗道/隐蔽系统 | 复杂侦察逻辑，后补 |
| 结盟外交 | 后续扩展 |
| 多地图/剧本 | 后续扩展 |

## 4. MVP 验收标准

MVP 必须满足以下全部条件才算通过：

- [ ] 2–4 人可完整进行一局（开局到分胜负）。
- [ ] 局前可配置装备，战备点 + 负重生效。
- [ ] 搜刮队可进入禁区镇搜刮并撤离。
- [ ] 边建造可在六边形边上放置并影响通行。
- [ ] 撤离路径经过敌方边触发劫掠战。
- [ ] 装备丢失惩罚生效（全灭/被劫丢装备）。
- [ ] 保险箱保留机制生效。
- [ ] 至少 2 条胜利条件可达成。
- [ ] 单局时长 60–90 分钟。
- [ ] 3 势力技能可正常使用。
- [ ] AI 蚀体可正常行动与战斗。

## 5. 后续扩展路线（优先级排序）

### 5.1 P1（MVP 后首批扩展）
- 扩展至 6 人 + 6 势力。
- 补全装备/建筑至完整列表。
- 实装四幕剧情 + 2–3 个全局事件。
- 实装科技树。

### 5.2 P2
- 实装士气系统。
- 实装暗道/伏击隐蔽系统。
- 补全 4 胜利条件。
- 增加人机 AI 强度档位。

### 5.3 P3
- 结盟/外交系统。
- 多地图/剧本。
- 赛季/排行榜。
- 观战/复盘系统。

## 7. 风险与应对

| 风险 | 应对 |
|---|---|
| 双玩法耦合不自然 | MVP 优先验证耦合，不自然则重构 |
| 单局时长过长 | 数字版加速结算；阶段并行（非交互阶段） |
| 装备丢失劝退 | 保险箱 + 基础装备保底 + 人机模式练手 |
| 6 人等待久 | 行动阶段限时 + 非自己回合可预规划 |
| 平衡难调 | 数据驱动配置 + 赛季迭代 |
